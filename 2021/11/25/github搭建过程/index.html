Posted on 2021-11-25

问题一

如何使用cmd进行node，git for windows，hexo，的安装

解决方法

node安装

先在cmd中找到要安装的地址，然后输入node -v命令即可

安装git for windows

先在cmd中找到要安装的地址，然后输入git命令即可

安装hexo

先在cmd中找到要安装的地址，然后输入npm install hexo-cli -g即可

问题二

如何用hexo搭建博客

在cmd中输入hexo init 博客名

然后进入博客目录输入npm install

然后就可以在本地进行运行，输入hexo s -p555
